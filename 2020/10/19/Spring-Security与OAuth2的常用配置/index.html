其他

令牌保存位置 TokenStore

。TokenStore 接口有多种实现类用来保存 access_token：

  • JDBCTokenStore：令牌保存到数据库中
  • InMemoryTokenStore：令牌保存到内存中
  • RedisTokenStore：保存到Redis
  • JwtTokenStore：使用jwt就是无状态登录，服务端不保存
  • JwkTokenStore：保存到 JSON Web Key 中
