Kafka介绍

Kafka是一种高吞吐量的分布式发布订阅消息系统，是消息中间件的一种。它一些基本术语：

  1. Kafka将消息以topic为单位进行归纳
  2. 向Kafka topic发布消息的程序称为producers
  3. 预定topic并消费消息的程序称为consumer
  4. Kafka以集群方式运行，可由一个或多个服务组成，每个服务叫做一个broker
