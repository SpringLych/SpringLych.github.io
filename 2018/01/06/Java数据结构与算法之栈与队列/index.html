栈

栈的定义

栈是一种特殊的线性表。其特殊性在于限定插入和删除数据元素的操作只能在线性表的一端进行。

栈的特点

  1. 后进先出（rear In front Out），简称为LIFO
  2. 限定只能在栈顶进行插入和删除操作。
  3. n个数依次入栈，则出栈方式共有C（2n,n）/(n+1)种

栈的应用

  1. 数值转换
  2. 表达式求值
  3. 浏览器历史纪录
  4. 解析xml/json等

队列

队列是一种特殊的线性表，只允许在表的前端（front）进行删除操作，在表的后端（rear）进行插入操作。

队列的特点

  1. 先进先出（First In First Out），FIFO
